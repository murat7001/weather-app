# REACT WEATHER APP 

You can find the live version [here](https://weather-app-hkj9.vercel.app/).

# Used Tools
- React
- Axios
- Context API

# Start
- cd weather-app
- Create .env folder
- Make sure to write REACT_APP_API_KEY=your_api_key in the .env folder. You can get the key from [here](https://www.weatherapi.com/)
- npm install
- npm start
